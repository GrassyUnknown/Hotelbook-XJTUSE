# 酒店管理系统

XJTUSE大三上2023-2024-1 软件项目管理综合训练

## 前端

前端采用vue2进行开发

确保安装了node.js（npm）后，使用终端在/vue文件夹内运行

```
npm run dev
```

即可使前端界面在8088端口运行

## 后端

后端采用Java Springboot进行开发

在IDEA项目中，需配置正确的maven路径和springboot运行配置

配置完成后，运行项目，后端将在8080端口运行

## 数据库

数据库采用mysql数据库，可以通过执行sql.sql直接创建表和导入测试数据
